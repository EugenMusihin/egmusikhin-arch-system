Тема: Реализация приложения для управления планами развития сотрудников
Директор компании хочет сделать процесс управления планами сотрудников централизованными, следовательно необходимо создать единую систему для управления индивидуальными планами развития, карьерными траекториями и оценками комптенций сотрудников.
Пользователи: директор, наставники и сотрудники компании
Требования: 
- Сотрудники могут формулировать цели и планы развития
- Наставники могут редактировать планы, оставлять комментарии и рекомендации
- Уведомление для пользователей о внесенных изменениях
- Сотрудники и наставники могут задавать компетенции в определенных навыках
- Возможность интеграции с существующими инструментами отчетности
Дополнительный контекст:
- Компании необходимо внедрить такое решенияm так как в данный момент времени подобного решения нет
- Компания насчитывает порядка 18 сотрудников
- В долгосрочной перспективе данная практика уменьшает время на ручное составление планов и унифицирует содержаение для каждого сотрудника
